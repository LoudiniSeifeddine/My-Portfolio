# My-Portfolio
🎨 Minimalist Developer Portfolio
A sleek, responsive portfolio website built with a focus on clean typography and interactive user experience. This project features a custom-built Dark Mode toggle and a unique dotted grid background rendered entirely in CSS.

🚀 Features
Dynamic Theme Switching: A custom-engineered Light/Dark mode toggle with smooth CSS transitions.

Modern Typography: Utilizes the Montserrat font family with precise letter-spacing and weighting for a high-end feel.

CSS-Only Grid System: A professional background created using radial-gradient instead of heavy image files to ensure fast load times.

Responsive Layout: Built using Flexbox for a structured and clean alignment of project cards and skill sets.

🛠️ Tech Stack
HTML5: Semantic structure for better accessibility.

CSS3: Custom properties, Flexbox, and complex gradients.

JavaScript (ES6): DOM manipulation for theme persistence and interaction logic.
